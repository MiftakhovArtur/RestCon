Я выполнил задание по добавлению REST API и переписыванию интерфейса на JavaScript. Вот что я сделал:

1. **Написал REST-контроллеры** для всех операций с пользователями:
   - GET /api/users - получение списка всех пользователей
   - GET /api/users/{id} - получение пользователя по ID
   - POST /api/users - создание нового пользователя
   - PUT /api/users/{id} - обновление существующего пользователя
   - DELETE /api/users/{id} - удаление пользователя
   - GET /api/roles - получение списка всех ролей

2. **Переписал весь интерфейс на JavaScript с использованием Fetch API**:
   - Таблица с пользователями теперь заполняется динамически через fetch запрос к REST API
   - Все модальные окна (добавление, редактирование, удаление) работают через JavaScript
   - Формы отправляются асинхронно без перезагрузки страницы
   - Добавил обработку ошибок и сообщения об успешных операциях

3. **Обеспечил отсутствие перезагрузок страницы**:
   - При создании пользователя таблица обновляется автоматически без перезагрузки
   - При редактировании данные сразу отображаются в таблице
   - При удалении строка исчезает из таблицы сразу после подтверждения
   - Модальные окна закрываются программно после успешных операций

В итоге получилось полноценное SPA-подобное приложение на чистом JavaScript без использования сложных фреймворков, где все операции с данными происходят асинхронно через REST API.
